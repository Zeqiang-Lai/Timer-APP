# Timer-APP

A Simple Stopwatch App for macOS.

<img src="img/preview12.png" alt="Preview" width="300"/>

## Usage

There are equvialent keyboard shortcuts for:

- LAP/RESET Button: `Alt` + `Space`
- START/STOP Button: `Space`
- List Button: `Alt` + `L`
